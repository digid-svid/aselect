ASelect module for Drupal

Author: Raymond Jelierse

Features
--------
 * Single sign-on via ASelect or OpenASelect with the ASelect Agent
 * Create new account for unmatched users.
 * Fallback account for unmatched users.
 * Disable password reset for regular user account.

Installation
------------
 * Install the module in your /sites directory.
 * Place the login directory in your site root.
 * Enable the module.
 * Configure the module's settings, if required.
